# Security policy

The hosted server accepts public HTTP and HTTPS website targets only. Private, local, loopback, link-local, and reserved network destinations are rejected before a tool fetch. Responses are bounded and tool calls are rate-limited.

The server exposes read-only analysis and offer-metadata tools. It has no browser, file, account, credential, purchase, or mutation capability.

Report a suspected vulnerability privately to `support@actablesite.com`. Do not include secrets or unrelated personal data. Include the affected tool, a minimal reproduction, and the security impact.
